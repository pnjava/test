Contact Center Call Flow

Employer/Member/Provider calls → AVAYA telecoms gateway/service centre (being replaced by Five9 — corporate directive upgrade) → routes calls (Aetna domain) to customer service agents. IVR (Edify) collects member ID etc., sends info to customer service; handles after-hours calls; talks to DG. Agents use Customer Service Interface (CSI Web) — web-based + desktop app — which reaches DG via WebDE.
