Meritain Claims Interface (MCI)

Streamlines the claims process on top of DG. Contractors use Claims UI (need less training); some functionality suppressed (e.g. denying claims); adds functionality not in DG (inventory management, assignment for supervisors). Built with LegaSuite (now Rocket Modern Experience) — GUI modernization/screen-scraping platform for green-screen apps; Build UI → deploy on Apache Tomcat. Meritain Operations team are the users. DG ↔ MCI bidirectional.
