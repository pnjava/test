# Meritain Platform Context Diagram
External users: prospect customers (marketing website), members (self-service website + IVR, health funds lookup, wellness offers; receive ID cards, communications, EOBs), brokers (broker portal; commission checks, eligibility/claims data, plan health reports), providers (paper claims in; checks, funds, EOBs, letters, provider portal), external auditors (auditor portal), plan sponsor back-office staff (batch enroll files, premium payment, enroll/update via fax/website; receive billed premium, reporting).
External partners: ChangeHealthcare/clearinghouses (837 in, 270/271, 276/277), BankTEC (paper claims → 837 + OCR image; member demographic), CMS (HIPAA eligibility request & COB claims extract / response), PBMs (claims; eligibility/accumulators/payments), medical management vendors/AHH (PreCerts; eligibility/claims), navigation partners Quantum/Accolade (claims/eligibility/accums for member services), reinsurance carriers (stop loss filing), US Bank (bank pay info — premium payment by credit card for COBRA; funds transfer), subrogation vendors PHIA (claims data out, response in).
Internal: Meritain Operations team → Meritain business operations.
