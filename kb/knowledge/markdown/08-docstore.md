# DocStore
Internally developed document management system. Stores DG static reports (monthly reports for plan sponsors, "Digital WIKI"), publishes custom content to MeritainConnect portal, provides static links externally for BPO plan documents. Interfaces: DG → DocStore; MeritainConnect via SOAP/REST; plan sponsor portals. Contact noted on diagram: Frank + "A N Other".
