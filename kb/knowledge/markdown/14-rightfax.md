# RightFax (COTS)
Inbound: Plan sponsor enrollment/eligibility admin faxes → RightFax → convert to doc → send to email group (fax attached) → manual entry into DG. Outbound: DG EOB via Fax Control Language, outbound faxes (logo etc.) to providers.
