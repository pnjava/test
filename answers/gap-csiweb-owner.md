# SME answer: Who owns CSI Web and what technology is it built with?

CSI Web is owned by Chris Byrd. It is built with ASP.NET on SQL Server and hosted on-premise. It is an existing application that needs enhancement for member preference capture (cell phone, SMS consent, RCS and email preferences), with preferences stored in the same local SQL Server database.
